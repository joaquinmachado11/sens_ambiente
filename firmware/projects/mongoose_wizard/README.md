# Mongoose Wizard Project

This project is generated by Mongoose Wizard, https://mongoose.ws/wizard/
Mongoose Wizard is a visual Web dashboard builder for embedded devices.

## Build and flash

Let's assume we're using ESP32C6 device:

```sh
$ idf.py set-target esp32c6
$ idf.py build
```


If you're using Docker, then:

```sh
$ docker run --rm -v `pwd`:`pwd` -w `pwd` espressif/idf idf.py set-target esp32c6
$ docker run --rm -v `pwd`:`pwd` -w `pwd` espressif/idf idf.py build
```

To flash, follow the instruction printed at the end of the build output.
